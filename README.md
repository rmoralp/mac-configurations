# mac-configurations

## Getting Started

### Automatically install all the things

```bash
bash ./commands/install.sh
```

It includes:

- Mac configurations
- CLI Tools
- Mac Apps
- Node & NVM
- VSCode extensions

### Manually set some configurations

Inside config folder, you can find some configurations for:
- VSCode
- NPM
- Git
- Oh my zsh
